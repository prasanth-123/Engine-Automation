To open in eclipse follow the steps:

1 -> find the DCDQ.7z zip file in the engine_automation folder.
2 -> extract the file in some path.
3 -> open eclipse, and create a new project and open the DCDQ project in eclipse.
4 -> take the jar files from "Engine_autoamation_Docs_jars" folder and add in your project's external library.

and that's it.

To Run :

1 -> open "Engine_automation_doc" excel file and follow the steps.
